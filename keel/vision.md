# sweech — Vision

The single tool every AI power user needs to manage multiple AI subscriptions without wasting quota.

## Core Promise
Switch between Claude Code, Codex, and 10+ AI providers seamlessly. Never waste expiring quota again.

## Surfaces
1. **CLI** (`sweech`) — Interactive launcher TUI, usage tracking, profile management
2. **SweechBar** (macOS menu bar) — At-a-glance usage, smart sort, one-click launch
3. **Federation API** (`sweech serve`) — HTTP API for integration with other tools

## Differentiators
- Smart sort that prioritizes expiring weekly quota
- AES-256 encrypted backups
- Shared data mode (symlinked profile dirs)
- OAuth token auto-refresh
- Real-time usage bars with burn-rate coloring
