# Sweech — Product Archaeology

## What It Was Meant To Be

**Switch between Claude Code, Codex, and 10+ AI providers seamlessly.**

Sweech is the CLI tool for managing multiple AI coding assistants simultaneously. Use Claude, Codex, Qwen, DeepSeek, OpenRouter, and local LLMs — all with different command names, all at the same time.

## Core Identity

- Multi-provider CLI management: install and alias multiple AI coding tools
- Simultaneous usage: run Claude on one task, Codex on another
- Provider abstraction: consistent interface across 10+ providers
- Zero-conflict setup: each provider gets its own command name

## The Gap It Fills

Developers want to use different AI tools for different tasks — Claude for architecture, a local model for quick edits, Codex for specific workflows. Sweech makes multi-provider usage frictionless instead of requiring manual PATH/alias management.

## Status

Active development. Core switching mechanism functional.
